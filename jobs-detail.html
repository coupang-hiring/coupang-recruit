大模型算法工程师 / 研究员（基础模型方向）

上海/北京
算法 📍 上海/北京 ⏱ 5-10年 🎯 P7 / P8
▌团队介绍

Coupang Foundation Model（CFM）团队是 Coupang 核心的基座大模型研发团队，致力于打造行业领先的电商垂直领域基础大模型，为 Coupang 全局业务提供 AI 能力支撑。模型全面覆盖商品理解、搜索、广告、客户服务等下游核心场景，并通过标准化 Token API 向下游提供高效、稳定、高质量的模型服务。

▌核心职责

深度参与 CFM 核心模型的迭代与创新，负责以下一个或多个方向的技术研发与突破：

  • 基模预训练 (Pre-training)：海量数据的清洗、配比与策略优化，探索高效的知识压缩与极致的模型底座能力

  • 计算机科学、人工智能或相关专业本科及以上学历
  • 深刻理解大语言模型底层结构，在预训练 / SFT / RL / Agent 招募方向中至少熟悉并深耕其中任意一项
  • 背景包容：欢迎来自专门基础模型团队或业务团队中负责算法研发的人才 —— 只要您对大模型训练方向有深入实践与见解，我们都非常期待
▌加分项
  • 在 ACL / NeurIPS / ICLR / ICML 等 NLP、机器学习领域顶级学术会议发表过高质量论文
  • 曾作为核心贡献者参与过知名开源 / 闭源大模型的研发，并拥有大模型 Technical Report 的署名或贡献经验
