Agentic AI Engineer

北京/上海
AI 📍 北京/上海 ⏱ 2年+
▌岗位介绍

我们正在招聘一位偏应用落地的 Agentic AI Engineer，负责将大模型与 Agent 技术真正应用到业务场景中。你将设计和开发能够自主理解需求、调用工具、执行多步任务的AI Agent，推动 Agentic AI 在 Testing 等实际场景中的落地与规模化使用。

▌核心职责
  • 设计并实现可落地的 AI Agent，用于自动化测试、流程执行与业务协作
  • 基于 LLM 构建具备 Memory、Tool Calling 和多步任务规划能力的 Agent 系统
  • 集成 LangChain / LangGraph / OpenAI API 等框架，快速交付可用产品能力
  • 构建稳定的 Agent 工作流 (Prompt, Tool, RAG, Evaluation, Monitoring)
  • 优化 Agent 的可靠性、成本、响应速度与安全策略
  • 与测试、产品、平台团队协作，推动 Agent 在真实业务场景中的使用与推广
  • 持续跟进 Agent 技术趋势，并将新能力转化为可落地方案
▌技术能力要求
  • 熟练使用 Python 进行工程开发
  • 掌握 LLM 相关技术：Prompt、RAG、Function Calling、Fine-tuning 等
  • 熟练使用 LangChain、LangGraph、OpenAI API、Hugging Face 等至少一种框架
  • 有 Agent / Workflow / 自动化系统开发经验（如 AutoGPT、CrewAI、ReAct 等）
  • 理解 API 集成、系统设计、日志监控与评估体系
▌经验要求
  • 2年以上 AI / 后端 / 自动化工程经验
  • 至少 1年 LLM 或 Agent 应用落地经验
  • 有测试自动化、DevOps、Workflow Automation 经验者优先
▌加分项
  • 有企业级 AI Agent 落地或内部效率工具开发经验
  • 熟悉强化学习、知识图谱或多 Agent 协作架构
  • 有AI产品从 PoC 到生产环境的完整交付经验
